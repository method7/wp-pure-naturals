Plugin that allows integration with fidelipay Payment Gateway

== Description ==

Process online payments with fidelipay using the hosted Payment Gateway or direct. To understand more; go to https://fidelipay.co.uk/ for more information.

== Installation ==

= Minimum Requirements =
* WooCommerce 2.1 or higher

There are two ways in which you can install the fidelipay Payment Gateway into your WordPress/WooCommerce website, Automatic and Manual.

= Option 1 - Automatic Installation =
This is the easiest way to install the fidelipay Payment Gateway plugin into your website. Simply log into your WordPress dahsboard and navigate to the Plugins menu, then click Add New.

In the search box, type fidelipay and click Search Plugins. Once you have found the fidelipay plugin you can install it by simply clicking Install Now.

= Option 2 - Manual Installation =
This option requires access to your servers directory.

1. Unzip and upload the plugin folder to your /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Go to WooCommerce -> Settings and click on the Checkout tab.
4. Find fidelipay in the Payment Gateways section
5. Click the settings button to configure and enable the gateway.
6. Click 'Save Changes'.
